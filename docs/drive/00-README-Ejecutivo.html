ZOE Net Gestión — Resumen Ejecutivo

Versión actual: 1.5.0 · Última actualización: 30 abril 2026

📌 Qué es

Plataforma integral de gestión empresarial para ZOE Net, proveedor de internet (ISP) en zona rural de El Seibo y La Romana, República Dominicana.

Cubre extremo a extremo: clientes, cobros mensuales, instalaciones técnicas, ventas de equipos, facturación, contabilidad (libro diario), inventario, revendedores y reportes financieros.

🏗 Stack técnico

  • Frontend: Next.js 14 (App Router) + TypeScript + Tailwind CSS
  • Backend: Supabase (Postgres + Auth + RLS + Realtime + Storage)
  • UI/Charts: Recharts, Lucide icons, Sonner toasts
  • State: Zustand con persist
  • Deploy: Vercel auto-deploy desde rama main
